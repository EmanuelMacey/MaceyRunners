# MaceyRunners

A full-featured delivery app built with React Native and Expo. It connects customers with drivers for on-demand delivery of items from local stores.

## Quick start

1. Clone the repository
2. Install dependencies: `npm install`
3. Create a `.env` (see `.env.example`) with your Supabase credentials
4. Start dev server: `npm run dev`

## Development

- Linting: `npm run lint`
- Build web: `npm run build:web`
- Build android: `npm run build:android`
