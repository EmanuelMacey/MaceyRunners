# Contributor Covenant Code of Conduct

We expect all contributors to adhere to a standard of respectful and welcoming behavior. Anyone found to be in violation of this code may be asked to stop contributing and may be removed from the project at the maintainers' discretion.

Be kind, be respectful, and be helpful.

For full details, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
